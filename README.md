# 🌌 Dark Puck - 2D Platformer Game

**Dark Puck**, Unity 6 (6000.3.9f1) kullanılarak geliştirilmiş, neon estetiğine sahip dinamik bir 2D aksiyon-platform oyunudur. Oyuncu, karanlık tünellerde fenerini kullanarak yolunu bulmalı ve zorlu düşmanları alt ederek ilerlemelidir.

## 🚀 Öne Çıkan Özellikler

* **Dinamik Işık Mekaniği:** Işığa duyarlı kapılar! Fenerinizi (`F` tuşu) tuttuğunuz sürece aktif kalan, ışık çekildiğinde ise kapanan akıllı kapı sistemleri.
* **Gelişmiş Düşman Yapay Zekası:**
    * **Patrol Robot:** Belirli bir rotada devriye gezen ve oyuncuyu gördüğünde fiziksel geri tepme (knockback) uygulayan kara birimleri.
    * **Attack Drone:** Havada süzülen (hover effect) ve oyuncuyu takip ederek lazer mermileri ateşleyen uçan birimler.
* **Fizik Tabanlı Oynanış:** Hasar alındığında karakterin savrulduğu gerçekçi geri tepme sistemi ve pürüzsüz karakter kontrolleri.
* **UI & Sağlık Sistemi:** Dinamik can barı, hasar anında yanıp sönme (invulnerability) efekti ve görsel geri bildirimler.

## 🛠 Kullanılan Teknolojiler

* **Motor:** Unity 6 LTS
* **Dil:** C# (Scripting)
* **Grafik:** 2D Pixel Art & Universal Render Pipeline (URP) Lighting
* **Versiyon Kontrol:** Git & GitHub

## 🎮 Nasıl Oynanır?

1.  **Hareket:** `A` - `D` veya Ok Tuşları
2.  **Zıplama:** `Space`
3.  **Feneri Aç/Kapat:** `F`
4.  **Amaç:** Karanlık tünellerdeki düşmanlardan kaçınarak veya fenerinizi kullanarak kapıları açıp ilerlemek.
